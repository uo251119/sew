Tim John Berners-Lee

Sir Timothy John Berners-Lee es un científico de la computación conocido por ser considerado el inventor de de la World Wide Web. Actualmente es el director del World Wide Web Consortium (W3C).

Proyectos actuales

Solid

Solid es uno de los proyectos actuales de Berners-Lee. Trata sobre la descentralización de la web, con el objetivo de lograr sistemas que "pertenezcan" más a los usuarios y con ello además mejorar la provacidad de la web.

El proyecto comenzó en 1989, como solución a problemas existentes en la web para el W3C. Problemas tales como el abuso de los datos por las empresas dueñas de los sitemas que almacenan dichos datos.

Recientemente, en cierta asignatura del grado de Ingeniería Informática de Oviedo esta tecnología fué utilizada como base para unas prácticas de laboratorio.

Web Semántica

La Web Semántica es una extensión de la World Wide Web cuyo objetivo es hacer que Internet sea legible por máquinas. Para este fin, se utilizan tecnologías como Resource Description Framework (RDF) o Web Ontology Language (OWL).

Mediante las tecnologías utilizadas en el proyecto, se quiere lograr la formalización de metadatos, como por ejemplo categorías de cosas, relaciones entre entidades o conceptos. Esta formalización provee grandes beneficios frente al estado actual de la web, que es muy heterogéneo.
